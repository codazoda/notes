# Blender

## View

- Middle button rotate view
- Shift + Middel button pan
- Z - Wireframe mode
- 5 (num) - Orthagraphic Mode

## Menu's

- T - Left Toolbar Menu
- N - Right Properties Menu

## Select

- A - Select All (Not Command-A)
- B - Box Select
- C - Circle Select
- Right Click - Cancel Selection Mode

## Transform Object

- Left Click - Confirm
- Right Click - Cancel

- G - Grab
  - X - X Axis
  - Y - Y Axis
  - Z - Z Axis
- R - Rotate
- S - Scale
- Alt+S - Undo Scale
- Alt+R - Undo Rotate
- Alt+G - Undo Grab
